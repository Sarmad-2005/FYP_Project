2.Expanded Use Case
UC1 – User Authentication & Role Management
Section Main
Details
Use Case
User Authentication & Role Management
Actors
Project Manager, Project Staff, Auditor
Purpose
To allow users to securely log in and access system features based on their role.
Overview
The user enters credentials. The system verifies login information and grants access to the appropriate dashboard according to the assigned role. In case of invalid credentials, an error is displayed.
Type
Primary and Essential



UC2 – Project Creation & Setup
Section Main
Details
Use Case
Project Creation & Setup
Actors
Project Manager
Purpose
To create a new project and initialize key parameters such as objectives, budget, and team.
Overview
The project manager selects “Create New Project,” enters the title, scope, timeline, and assigns members. The system saves details, generates a project ID, and sets up initial milestones and budget templates.
Type
Primary and Essential
